Sturdy Components

Easily CDN-includable, Bootstrap-friendly web components written in HTML, CSS and JS. Bootstrap not required!

Hakimel Buttons

Mouse direction based button animations.
Try it out below!
Copy and include in your HTMl Document. 
<link rel="stylesheet" href="https://blumelume.github.io/sturdy-components/hakimelButtons/hak-btn-default.css">
<script type="text/javascript" src="https://blumelume.github.io/sturdy-components/hakimelButtons/hak-btn.js"></script>
For usage info visit the Github-Page
Hakim El Hattab developed reveal.js and founded Slides.com. When visiting his website I noticed the awesome buttons that he had there (they are also used in the Revealjs documentation).

www.hakim.se
Visit Him Online
I wanted to build those and share my solution with you.
Like everything in this project, they are easily included and follow common Bootstrap-like syntax. They are almost fully customizable through bootstrap-classes (color, text-alignement, etc.) and come in a standard Bootstrap color as well. Anything that can't be done through plain Bootstrap, can be adjusted in the hak-btn.js file - feel free to tinker.
